# VioChat
VioChat - simple lightweight minecraft 1.8+ chat utility (server plugin). 

#### features
- local chat (defaults to 100 blocks radius)
- roleplay chat
- admin chat
- global chat
- private chat (p2p)
- automatic messaging

[DOWNLOAD](https://github.com/violanes/VioChat/blob/master/VioChat.jar?raw=true)

> #### Required:
> * Vault (latest build)
> * Spigot / bukkit (1.8+)
