# VioChat
##### VioChat - simple minecraft plugin for chatting in 3 different channels: local ( 100 blocks ), global and admin ( special permission required ). Displays prefixes, suffixes and whatever you want. 

> #### Required:
> * Vault ( latest build )
> * Spigot / bukkit ( 1.8 + )
